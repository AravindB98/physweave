physweave

Weaving physics simulation and machine learning — differentiable simulation and neural PDE surrogates in one toolkit.

View on GitHub ▶ Try the Demo

🧒 What is this? (Plain English)

Physics simulators and machine learning, woven together: the simulator can 'feel' how changing an input changes the outcome (differentiable), and neural networks learn to imitate slow physics calculations so they run thousands of times faster.

✨ Key Features

Differentiable simulation

Gradients flow through physics, enabling optimization and control.

Neural PDE surrogates

Networks that imitate slow solvers at 1000× speed.

Modular design

Compose simulators, surrogates, and training loops freely.

Physics-informed ML

Blend physical laws with learned components.
